# NutriTrack - Calorie & Nutrition Tracker

NutriTrack is a console-based C++ application that helps users track calories, protein, carbs, fat intake, and BMI.

## Features
- User Registration & Login
- Meal Tracking
- Calorie Counter
- Nutrition Report
- BMI Calculator
- File Handling
- Daily Goal Tracking

## Technologies Used
- C++
- OOP Concepts
- File Handling
- Functions
- Switch Case

## Future Improvements
- Exercise Tracker
- Food Search
- Better UI
- Password Protection
