NutriTrack
A console-based nutrition and fitness tracking application built with C++.
NutriTrack allows users to create an account, track meals and nutrition, record exercise, calculate BMI, and save nutrition session history using file handling.

-Features

* User registration and login
* User profile management
* Custom daily calorie goals
* Food database loaded from a file
* Search food by name or number
* Add meals with quantity
* Track:
  * Calories
  * Protein
  * Carbohydrates
  * Fat
* View today's meal history
* Remove meals
* Exercise tracking
* Calories burned calculation
* Exercise summary
* Daily nutrition report
* Net calorie calculation
* BMI calculator
* Save nutrition session history
* View previously saved history
* Input validation for common invalid inputs
